# 우리의 여행 일기 💗

함께 다녀온 날들을 **지도**와 **달력**에 기록하는 커플 일기장 웹앱이에요.
서버 없이 동작하고, GitHub Pages에 그대로 올려서 쓸 수 있습니다. 모바일에 최적화되어 있어요.

## ✨ 기능
- **지도(홈)** — 처음엔 부산이 보이고, 기록이 쌓이면 다녀온 곳들이 모두 보이도록 자동으로 맞춰져요. 방문지는 하트 핀으로 표시되고, 핀을 누르면 그날 일기로 이동.
- **달력** — 만난 날짜에 마스코트/점이 표시되고, **날짜를 누르면 바로 사진 업로드·글 작성 편집기**가 열려요.
- **모아보기** — 모든 기록을 사진 카드로 최신순으로 볼 수 있어요.
- **다녀온 곳 기록** — 식당·카페·명소의 **위치(지도 선택) + 이름 + 주소 + 사진**을 저장.
- **사진 기록** — 하루에 여러 장, 자동으로 압축해서 저장.
- **함께한 날수(D-day)** — 설정에서 처음 만난 날을 넣으면 상단에 표시.
- **PWA** — 모바일에서 "홈 화면에 추가" 하면 앱처럼 실행돼요.
- **백업/복원** — 설정에서 파일로 내보내고 다른 기기에서 불러올 수 있어요.

## 🗂 데이터는 어디에 저장되나요?
기록은 **사용하는 기기(브라우저)** 안에 저장돼요 (IndexedDB / localStorage). 서버로 보내지 않아 안전하지만,
- 브라우저 데이터를 지우면 사라질 수 있어요.
- 다른 기기와 자동으로 동기화되지 않아요.

👉 그래서 가끔 **설정 → 백업 내보내기**로 파일을 저장해두세요. 새 기기에서는 **불러오기** 하면 그대로 옮겨져요.

## 🚀 GitHub Pages에 올리기
1. 이 폴더 전체를 GitHub 저장소에 올려요. (예: 저장소 이름 `our-diary`)
2. GitHub 저장소 → **Settings → Pages** 로 이동.
3. **Source** 를 `Deploy from a branch`, 브랜치는 `main`, 폴더는 `/ (root)` 로 설정하고 저장.
4. 잠시 후 `https://<아이디>.github.io/our-diary/` 주소로 열려요. 📱

> 모든 경로가 상대경로라 하위 폴더(`/저장소이름/`)에서도 잘 동작해요.
> 스마트폰에서 그 주소를 연 뒤 브라우저 메뉴의 **"홈 화면에 추가"** 를 누르면 앱처럼 쓸 수 있어요.

## 🎨 커스터마이즈
- **제목 / 처음 만난 날**: 앱 우측 상단 ⚙ 설정에서 바로 변경.
- **색/폰트**: `css/style.css` 상단 `:root` 변수(`--pink-400` 등)만 바꾸면 전체 테마가 바뀌어요. 폰트는 SUIT.
- **마스코트/로고/파비콘**: `js/mascots.js`, `assets/favicon.svg`, `assets/logo.svg`.
- **첫 지도 위치**: `js/map.js` 의 `BUSAN` 좌표와 `initHome`의 `zoom` 값.

## 🗺️ 카카오 지도 켜기 (선택)
기본은 무료·무설정 지도예요. **한국 지도앱처럼 보이게 + 가게 이름 검색**을 쓰려면 무료 카카오 키를 넣으세요.

1. [Kakao Developers](https://developers.kakao.com) 로그인 → **내 애플리케이션 → 애플리케이션 추가하기** 로 앱을 하나 만들어요.
2. 만든 앱 → **앱 키** 에서 **JavaScript 키** 를 복사.
3. 같은 앱 → **플랫폼 → Web 플랫폼 등록** 에 사이트 주소를 등록해요:
   - 배포 주소: `https://<아이디>.github.io` (본인 GitHub Pages 주소)
   - 로컬에서도 테스트한다면 `http://localhost:5321` 도 함께 등록
4. 앱의 **⚙설정 → 지도(선택) → 카카오 지도 키** 칸에 복사한 **JavaScript 키**를 붙여넣고 저장 → 자동 새로고침되며 카카오맵으로 바뀌어요.

이제 “다녀온 곳 추가” 시 **가게 이름으로 검색**하면 위치·이름·주소가 한 번에 채워져요.

> ⚠️ 3번의 **Web 플랫폼(도메인) 등록**을 빠뜨리면 지도가 회색으로 비어 보여요. 꼭 등록해 주세요.
> 키를 비우면 언제든 기본 지도로 돌아갑니다.

## 🛠 로컬에서 미리보기
`index.html`을 더블클릭해도 되지만, 일부 기능(사진 저장 등)은 간단한 로컬 서버에서 더 안정적이에요.
```bash
# Node.js가 있다면 (별도 설치 없이)
node .claude/serve.js 5321
# 그런 다음 http://localhost:5321 접속
```
> `.claude/` 폴더는 로컬 미리보기용 도구라 GitHub에 올리지 않아도 됩니다.
